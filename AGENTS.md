# Floor Haven — инструкции для AI

Проект: сайт компании «ПолМастер» (напольные покрытия). React + Vite + TypeScript + Tailwind + shadcn/ui.

## При работе в этом репозитории

1. **Правила** в `.cursor/rules/` задают соглашения по стеку, React/TS, компонентам и страницам — соблюдай их при правках и добавлении кода.
2. **Контент и UI** — на русском; имена переменных, функций и комментарии — на английском.
3. **Структура:** страницы в `src/pages/`, общие компоненты в `src/components/`, данные и типы в `src/data/`. Компоненты из `src/components/ui/` (shadcn) не меняй без необходимости.
4. **Стили:** используй готовые классы из `src/index.css` и Tailwind; новые глобальные классы добавляй в `index.css`.
5. **Навигация:** внутренние переходы — через `<Link to="...">` (react-router-dom).

При добавлении фич (новые страницы, формы, API) сохраняй текущий стиль кода и структуру проекта.
